# 財經終端：金控股息試算系統 - 完整說明

## 📘 系統概述

本系統專為台灣金控股投資人設計，整合「二代健保補充保費」與「所得稅可抵減稅額」計算邏輯，提供精準的稅後實領股息預估。

## 🧮 核心計算公式

### 1. 股息所得計算
```
預估年度配息總額 = 持股數量 × 每股股息
```
- **每股股息**: 根據各金控最近一年實際配息記錄
- **持股數量**: 以「股」為單位輸入（1張 = 1000股）

### 2. 二代健保補充保費

**法源依據**: 全民健康保險法第 31 條

**觸發條件**: 單次股利所得 ≥ NT$ 20,000

**費率**: 2.11%

**計算公式**:
```
補充保費 = {
  配息總額 × 2.11%,  if 配息總額 >= 20,000
  0,                  if 配息總額 < 20,000
}
```

**範例計算**:
- 富邦金持股 10,000 股，股息 NT$ 3.0
- 配息總額 = 10,000 × 3.0 = NT$ 30,000 ✓ (超過門檻)
- 補充保費 = 30,000 × 2.11% = **NT$ 633**

### 3. 所得稅可抵減稅額

**法源依據**: 所得稅法第 126 條（股東可扣抵稅額）

**抵減率**: 8.5%

**每戶上限**: NT$ 80,000

**計算公式**:
```
可抵減稅額 = MIN(配息總額 × 8.5%, 80,000)
```

**機制說明**:
當您的股息收入被列入綜合所得稅時，政府允許您從應納稅額中扣除已繳納的營利事業所得稅（公司股東可扣抵稅額），避免重複課稅。

**範例計算**:
- 配息總額 NT$ 30,000
- 可抵減 = 30,000 × 8.5% = **NT$ 2,550**
- 若配息達 NT$ 941,176，則 941,176 × 8.5% = 80,000 (觸及上限)

### 4. 稅後淨得計算

**Step 1: 扣除二代健保**
```
淨額 = 配息總額 - 補充保費
```

**Step 2: 計入綜合所得稅**
```
所得稅負 = 配息總額 × 個人邊際稅率
```

**Step 3: 計入可抵減稅額**
```
真金白銀收益 = 淨額 - 所得稅負 + 可抵減稅額
```

## 📊 實際試算範例

### 情境一：小額投資人
- **標的**: 兆豐金 (2886)
- **持股**: 5張 = 5,000 股
- **股息**: NT$ 1.8 / 股
- **邊際稅率**: 5%

| 項目 | 計算 | 金額 |
|------|------|------|
| 預估配息 | 5,000 × 1.8 | NT$ 9,000 |
| 二代健保 | 未達門檻 | NT$ 0 |
| 所得稅負 | 9,000 × 5% | NT$ 450 |
| 可抵減稅額 | 9,000 × 8.5% | NT$ 765 |
| **實領收益** | 9,000 - 0 - 450 + 765 | **NT$ 9,315** ✅ |

> 💡 **關鍵洞察**: 在低稅率級距下，可抵減稅額甚至超過所得稅負，產生「稅務溢價」效應。

### 情境二：中大額投資人
- **標的**: 富邦金 (2881)
- **持股**: 30張 = 30,000 股
- **股息**: NT$ 3.0 / 股
- **邊際稅率**: 20%

| 項目 | 計算 | 金額 |
|------|------|------|
| 預估配息 | 30,000 × 3.0 | NT$ 90,000 |
| 二代健保 | 90,000 × 2.11% | NT$ 1,899 |
| 所得稅負 | 90,000 × 20% | NT$ 18,000 |
| 可抵減稅額 | 90,000 × 8.5% | NT$ 7,650 |
| **實領收益** | 90,000 - 1,899 - 18,000 + 7,650 | **NT$ 77,751** |

**實質稅率**: (1,899 + 18,000 - 7,650) / 90,000 = **13.6%**

## 🔍 節稅策略建議

### 策略一：避開補充保費門檻
如果預估配息接近 NT$ 20,000，考慮：
- 分散至家人名下
- 調整持股數量至 19,999 以下

### 策略二：利用可抵減上限
- 最佳配置區間：配息總額約 NT$ 800,000 - 900,000
- 此時抵減效能最大化（接近 80,000 上限但不浪費）

### 策略三：邊際稅率優化
- 若您處於 5% 或 12% 級距：股息投資極度有利
- 若處於 30% 或 40% 級距：建議評估資本利得策略

## 📱 系統功能說明

### 快捷鍵設計
系統提供「張」為單位的快捷輸入，符合台灣投資人習慣：
- **1張** = 1,000 股
- **5張** = 5,000 股
- **10張** = 10,000 股
- **50張** = 50,000 股

### 敏感度分析儀
系統自動計算不同邊際稅率下的實質收益，協助您評估最佳配置：
- 5% 級距（年所得 ≤ 56萬）
- 12% 級距（56萬 - 126萬）
- 20% 級距（126萬 - 252萬）
- 30% 級距（252萬 - 472萬）
- 40% 級距（≥ 472萬）

## ⚠️ 重要提醒

1. **本系統僅供參考**: 實際稅負以國稅局核定為準
2. **股息非保證**: 配息金額會因各金控獲利狀況調整
3. **稅率可能變動**: 政府政策調整時需重新試算
4. **其他所得計入**: 您的其他收入會影響邊際稅率
