# 金融戰情室 (Financial Monitor) 當前需求與功能摘要

本文件摘要目前「金融戰情室」應用程式的功能與實作狀態，供確認後再進行後續修改。

## 1. 核心功能 (Core Features)

### 1.1 市場數據監控 (Market Data Monitoring)
- **即時追蹤**: 監控 12 檔主要金融股（如富邦金、國泰金、中信金等）。
- **數據來源**: 
    - 優先從 Google Finance 抓取即時報價 (Price, Change, Change%)。
    - 若 API 失效，自動切換至模擬數據 (Simulation Mode) 以維持介面運作。
- **重新整理**: 每 3 秒自動更新一次數據。

### 1.2 個股走勢圖 (Stock Chart)
- **視覺化**: 使用 `Chart.js` 顯示個股的當日走勢線圖。
- **數據累積**: 具備 `priceHistoryStore` 機制，在市場開盤期間持續累積個股價格點，以形成平滑的線圖。
- **切換顯示**: 點擊左側列表或強/弱勢股列表，可切換圖表顯示的目標個股。

### 1.3 強弱勢股排名 (Top Movers)
- **強勢股**: 自動計算並顯示前 3 名漲幅最大的個股。
- **弱勢股**: 自動計算並顯示前 3 名跌幅最大的個股。
- **互動性**: 點擊排名中的個股可連動更新走勢圖與新聞。

### 1.4 新聞整合 (News Integration)
- **智能搜尋**: 從 Google News 抓取相關新聞。
- **連動機制**:
    - 預設顯示一般金融新聞 (關鍵字：金控)。
    - 選取個股後，自動切換為該個股的相關新聞。
- **顯示資訊**: 顯示新聞標題與來源，點擊可開啟原始文章。

## 2. 介面與體驗 (UI/UX)

- **現代化設計**: 採用深色模式 (Dark Mode) 為主的儀表板介面，搭配毛玻璃效果 (Glassmorphism)。
- **響應式佈局**: 左側導覽列 (Ticker List) 配合右側主顯示區。
- **主題切換**: 具備樣式切換功能 (ThemeSelector)。
- **圖示庫**: 使用 `lucide-react` 提供清晰的視覺指引。

## 3. 技術架構 (Technical Stack)

- **前端框架**: React 19 + Vite。
- **樣式管理**: Tailwind CSS。
- **動畫效果**: Framer Motion。
- **圖表庫**: Chart.js / react-chartjs-2。
- **Proxy 機制**: 透過 Vite Config 設定 `/api/google` 與 `/api/gnews` 的代理，解決 CORS 問題並進行網頁抓取。
